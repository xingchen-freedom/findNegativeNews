# findNegativeNews
First internship assignment
# Content
1. 使用apache httpclient建立了一个http服务器，地址为 "http://localhost:8800/baidunews"
2. 项目抓取百度新闻首页的热门新闻板块，将该板块的新闻标题放入情感计算 API 得到正负面评价。
3. 将负面结果的新闻标题和URL保留，并通过http服务器对外展示。
